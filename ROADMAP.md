# Roadmap

This roadmap defines **intentional phases**. Skipping phases is discouraged.

---

## Phase 0 — Bootstrap (DONE)

- Monorepo structure
- Melos + Pub workspace
- VS Code workspace
- CI-ready commands
- Internal-only packages

---

## Phase 1 — Foundations (CURRENT)

Goal: Establish a stable visual language.

Deliverables:

- Design tokens (color, spacing, typography, radius)
- Theme infrastructure
- Light/Dark theme support
- Token preview in showcase

---

## Phase 2 — Core Components

Goal: Validate the system with high-impact components.

Initial components:

- Button
- Input / TextField
- Card / Surface

Focus:

- States (hover, pressed, disabled)
- Accessibility
- Consistent API surface

---

## Phase 3 — Expansion

- Layout primitives
- Feedback components (toast, alert)
- Icons integration
- Motion tokens

---

## Phase 4 — Stabilization

- API freeze
- Documentation hardening
- Versioning strategy
- Optional publication strategy

---

## Phase 5 — Experimental

- Alternative themes
- Opinionated presets
- Platform-specific variants
